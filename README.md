# jumpboy

## Overview
- This is 2D game application generated by pyxel.

## Environment
- Python 3.12
- pyxel
- python-dotenv
- pyxel-universal-font
- PixelMplus

## Getting Start
```bash
pip install -r requirement.txt
cp .env.sample .env
pyxel run app.py
```

## Edit Asset File
```bash
pyxel edit assets/jumpboy.pyxres
puf edit
# Copy from all file in 'assets/font' folder.
```
