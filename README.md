# jumpboy

## Overview
- This is 2D game application generated by pyxel.

## Environment
### Software language
- Python 3.12

### Library
- pyxel
- python-dotenv
- pyxel-universal-font

### Font
- PixelMplus

## Getting start
```bash
# Install library.
pip install -r requirement.txt

# Create environment file.
cp .env.sample .env

# Start game.
pyxel run app.py
```

## Asset file edit
```bash
# Edit pyxel assets.
pyxel edit assets/jumpboy.pyxres

# Edit font.
puf edit
# Copy font files in 'assets/font' folder.
```
