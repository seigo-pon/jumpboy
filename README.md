# jumpboy

## Overview
- This is 2D game application generated by pyxel.

## Environment
- Python 3.12
- pyxel
- python-dotenv

## Getting Start
```bash
pip install -r requirement.txt
pyxel run app.py
```

## Edit Asset File
```bash
pyxel edit assets/jumpboy.pyxres
```
